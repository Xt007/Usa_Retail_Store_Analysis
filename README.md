# Usa_Retail_Store_Analysis

## Project Overview

This project is a SQL-based exploratory data analysis (EDA) of an e-commerce retail dataset using Google BigQuery.

The goal is to analyze customer behavior, sales trends, delivery performance, payment patterns, and freight costs to generate useful business insights.

---

## Objectives

- Perform exploratory data analysis (EDA)
- Analyze customer distribution across cities and states
- Identify yearly and monthly order trends
- Study customer purchasing behavior by time of day
- Analyze sales and freight performance
- Evaluate payment methods and installment patterns
- Measure delivery performance against estimated delivery dates

---

## Dataset

The dataset includes the following tables:

- Customers
- Orders
- Order Items
- Payments
- Products
- Sellers
- Reviews
- Geolocation

---

## Tools Used

- Google BigQuery
- SQL
- GitHub

---

## Key Insights

- Seasonal trends were observed in orders
- Customer distribution varies across states
- Delivery performance differs by region
- Installment payments are commonly used
- Freight costs vary across locations
